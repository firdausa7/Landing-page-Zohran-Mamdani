# Landing-page
## Features

### 1. Responsive Landing Page
- Semantic HTML structure
- CSS Grid and Flexbox layouts
- Mobile-first responsive design
- Smooth scrolling navigation
- CSS animations

### 2. Interactive Form
- Real-time form validation
- Error messages without page reload
- Professional styling
- Success feedback

### 3. Counter Application
- DOM manipulation
- Event listeners
- Visual feedback
- Keyboard support

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Variables)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (via CDN)

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Setup Instructions

1. Clone or download the project files
2. Ensure all file paths are maintained as in the structure above
3. Open `index.html` in a web browser
4. For development, use a local server for best performance

## Development Notes

- CSS uses CSS Custom Properties for theming
- JavaScript is modular and separated by functionality
- Responsive design uses mobile-first approach
- Form validation provides real-time user feedback
